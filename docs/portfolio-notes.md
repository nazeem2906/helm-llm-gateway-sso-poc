# Portfolio Notes

This project demonstrates:

- Helm chart authoring
- Kubernetes deployment patterns
- SSO integration with Keycloak
- LDAPS federation design
- LLM API gateway deployment
- PostgreSQL-backed application state
- Prometheus metrics exposure
- Secure configuration through values and Kubernetes Secrets

## Why this is anonymized

The project intentionally uses:

- example.com
- example.local
- placeholder LDAP DNs
- placeholder secrets
- generic model names
- generic group names

Do not commit:

- real production values
- real certificates
- real tokens
- internal DNS names
- internal IP addresses
- company-specific LDAP structure
